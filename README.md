# simple_face_detection--人脸测试程序
* ## 这个项目是基于Python2.7.14 且 OpenCV库是3.1.0 实现的小程序。
* ## 预期实现的目标：能够在指定的几个人中，实时检测并识别其中任意一个人；在实时检测的过程中，能够将所识别到的人脸框起来，并在框上显示其检测到人物对应的姓名。</br>
